# Stack Overflow Search — QA Automation Suite

## 📖 Overview
Automated test suite for **Stack Overflow** search and query functionality using Selenium WebDriver, Java, and TestNG. Validates search workflows, tag-based filtering, and result validation on one of the world's largest developer Q&A platforms.

---

## 🚀 Key Testing Highlights
- **Search Validation:** Verified keyword and tag-based search returns relevant results
- **Tag Filter Testing:** Validated filtering by programming language and framework tags
- **Sort Validation:** Tested sorting by relevance, votes, and newest
- **Negative Testing:** Handled empty queries and special character inputs
- **POM Architecture:** Clean separation of page interactions and test logic

---

## 🛠️ Tech Stack
| Technology | Purpose |
|---|---|
| Java | Core language |
| Selenium WebDriver | Browser automation |
| TestNG | Test runner & assertions |
| Maven | Build & dependency management |

---

## ▶️ How to Run
```bash
git clone https://github.com/ritvikz/crio-stackoverflow-qa.git
cd crio-stackoverflow-qa
mvn test
```
